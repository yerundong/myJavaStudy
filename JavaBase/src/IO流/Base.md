# 流的操作流程：
1. File对象实例化；
2. 流实例化，创建流；
3. 流的操作（读、取...）；
4. 关闭流。
- 注意：不同的流操作，1、4步操作相同，2、3步不同。

# 结构
```
├──节点流
  └──文件流
    ├──FileInputStream：字节输入文件流
    ├──FileOutputStream：字节输出文件流
    ├──FileReader：字符输入文件流
    └──FileWriter：字符输出文件流
```
# 分类
### 字节流
传输过程中，传输数据的最基本单位是字节的流。一般以stream结尾。
### 字符流
传输过程中，传输数据的最基本单位是字符的流。
